#  AI Diet Planner

An AI-powered diet planning application that generates personalized, structured diet plans based on user inputs such as age, weight, and medical conditions. Built with modern frontend technologies and integrated with Google's Gemini API.


## 🚀 Features

-  Generate personalized diet plans using AI
-  Clean and responsive UI (Tailwind CSS)
-  Real-time API integration with Gemini
-  Form validation and error handling
-  Loading states with spinner feedback
-  Secure API key handling via environment variables
-  Dynamic HTML rendering of AI responses

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **HTTP Client:** Axios
* **Notifications:** React Toastify
* **Icons:** Lucide React
* **AI Integration:** Google Gemini API


---

## 🔐 Security Note

⚠️ Never expose your API key in frontend code.
Always store it in `.env` or use a backend proxy for production.

---

## 📸 How It Works

1. User enters age, weight, and medical condition
2. Data is sent to Gemini API
3. AI generates a structured diet plan (HTML format)
4. Response is rendered dynamically in UI
