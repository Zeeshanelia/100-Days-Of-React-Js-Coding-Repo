# Context Api Functional Website
 
Mini E-commerce Project - Learning Summary

- Built a dynamic e-commerce app using React.

- Implemented a functional shopping cart with add, remove, increment, and decrement features.

- Managed global state with React Context API.

- Created a custom hook (useCart) for easy context access.

- Learned React Router for multi-page SPA navigation (/, /cart, /about).

- Styled components with Tailwind CSS for rapid prototyping.

-  Applied responsive design using grid layouts for products.

- Added smooth hover effects for product cards and buttons.

- Handled price calculation errors (NaN) by using numeric values.

- Fixed runtime errors like “useContext must be used inside provider”.

- Used console logs and state inspection for debugging issues.

- Learned project structure best practices: components, context, pages.

- Separated concerns in files: App.jsx, main.jsx, context providers.

- Maintained clean folder hierarchy for scalability.

- Implemented dark mode toggle with Context API.

- Used localStorage to persist dark mode preference.

- Displayed dynamic cart badge updating with cart item count.

- Integrated icons (lucide-react) for better UI/UX.

- Learned to combine Tailwind + React effectively.

- Gained practical experience in building modern, maintainable React apps.
