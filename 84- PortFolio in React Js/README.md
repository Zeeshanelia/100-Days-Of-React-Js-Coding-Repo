#  Personal Portfolio in React Js

A modern, responsive **developer portfolio website** built with React to showcase projects, skills, and contact information. Designed with smooth UX, clean UI, and performance-focused practices.

---

## 🛠 Tech Stack

* **React 18** – Component-based UI
* **Axios** – API requests handling
* **React Hook Form** – Efficient form management & validation
* **React Hot Toast** – Elegant notifications
* **React Icons** – Icon library for UI enhancement
* **React Scroll** – Smooth scrolling navigation
* **React Typed** – Typing animation effects

---

## ✨ Features

* ⚡ Fully responsive design (mobile + desktop)
* 🎯 Smooth scrolling navigation
* ⌨️ Typing animation for hero section
* 📩 Contact form with validation
* 🔔 Toast notifications for user feedback
* 🌐 API integration using Axios
* 🎨 Clean and modern UI

---

## 📌 Usage

* Update personal information in components
* Replace project data with your own work
* Connect contact form with your backend or email service
* Customize styling as needed

---

## 📸 Key Sections

* **Hero Section** – Introduction with typing effect
* **About** – Brief profile summary
* **Projects** – Showcase of work
* **Skills** – Technologies you use
* **Contact** – Functional form with validation
