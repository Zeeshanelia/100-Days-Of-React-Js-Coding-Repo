# Attendance Tracker

A React-based attendance tracking app built with **Ant Design**, **Zustand**, and **Moment.js**.

## Features
- ✅ Check-in / Check-out with live HH:MM:SS timer
- 📅 Calendar view showing daily attendance with duration tags
- 🔒 Prevents duplicate sessions — only one check-in per day allowed

## Tech Stack
React • Ant Design • Zustand • Moment.js • nanoid • animate.css

## Getting Started
Clone the repo, run `npm install`, then `npm start` to launch the app.
